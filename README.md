# Teyzix Core | SaaS Landing Page & Onboarding Portal

A modern, responsive, and interactive frontend application designed for a SaaS product. This project fulfills the requirements of the Teyzix Core Internship (June Batch).

## 🚀 Features
- **Modern Landing Page**: Hero section, Product Overview, Features, Pricing, and FAQ sections.
- **Customer Onboarding Portal**: Multi-step registration form with real-time validation and progress tracking.
- **Dashboard Preview**: Sleek analytics view for system monitoring.
- **Customization**: Dark/Light mode toggle, Accent color themes (Emerald, Blue, Violet), and Internationalization (English/Spanish).
- **Responsive Design**: Mobile-first architecture optimized for all devices.

## 🛠 Tech Stack
- **Framework**: React.js
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Icons/UI**: Custom modern design components
